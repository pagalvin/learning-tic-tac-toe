# Tic-Tac-Toe (TypeScript)

A browser-based tic-tac-toe game with one human player (`X`) versus a medium-strength computer player (`O`).

## Features

- Single-round play with a restart button
- Human vs computer turns
- Medium AI behavior (uses obvious win/block moves often, but still makes occasional mistakes)
- TypeScript + Vite setup for fast local development

## Getting started

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open the local URL shown in the terminal.

## Build and preview

1. Create a production build:
   `npm run build`
2. Preview the production build locally:
   `npm run preview`
